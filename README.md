# UZS FX Monitor

Premium currency dashboard web application with real-time exchange rates.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Real-time currency exchange rates
- Pin/unpin favorite currency pairs
- Multi-language support (EN/RU/UZ)
- Dark/Light theme switching
- Currency converter
- Responsive design
- Smooth animations
